# gitlab-00-tutorial

## The Issue
This is a tutorial — no issue to debug. Just run `gitlab-ci-local` to see the pipeline execute.

## Solution
```bash
gitlab-ci-local
```

## Understanding
GitLab CI/CD pipelines are defined in `.gitlab-ci.yml` files. The `gitlab-ci-local` tool lets you run them locally without a GitLab instance.
